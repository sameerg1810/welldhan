I am building WELLDHAN — a wellness + organic food platform for gated communities.

TECH STACK:
- Frontend: Vite + React.js + TypeScript + Material UI (MUI v5)
- Backend: FastAPI + MongoDB (already built — auth routes exist)
- State: React Query (TanStack Query) for server state + Zustand for auth/UI state
- HTTP: Axios with interceptors for JWT token
- Routing: React Router v6
- Forms: React Hook Form + Zod validation
- Theme: Dark green (#1a4d2e primary, #2d7a47 secondary, #4ade80 accent)

EXISTING AUTH (already done in backend):
- POST /auth/login → email + password → returns challenge_id + masked phone
- POST /auth/send-otp → phone → sends OTP via Gmail SMTP
- POST /auth/verify-otp → phone + otp → returns { token, role, user_id, user_data }
- GET /auth/me → returns logged-in user data
- Roles: User, Trainer, Manager, Admin

export const useToggleFood = () => useMutation(foodApi.toggleItem, {
  onMutate: async (newData) => {
    // Optimistic update — immediately flip the toggle in cache
    await queryClient.cancelQueries(['food', 'preferences'])
    const prev = queryClient.getQueryData(['food', 'preferences'])
    queryClient.setQueryData(['food', 'preferences'], (old) => updateToggleInCache(old, newData))
    return { prev }
  },
  onError: (err, newData, context) => queryClient.setQueryData(['food', 'preferences'], context.prev),
  onSettled: () => queryClient.invalidateQueries(['food', 'preferences']),
})
